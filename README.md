# Super Bingo Game

## Introduction
The Super Bingo Game is an exciting digital adaptation of the classic bingo game, designed for players of all ages. This documentation outlines the key features, requirements, and instructions to set up and play the game.

## Features
- **User-friendly Interface**: A simple and intuitive design to enhance the playing experience.
- **Multiplayer Support**: Play with friends or join random games.
- **Customizable Cards**: Create bingo cards with your preferred numbers.
- **Real-time Chat**: Communicate with other players during the game.
- **Statistics Tracking**: Keep track of your game history and performances.

## Requirements
### Software Requirements
- **Operating System**: Windows, macOS, or Linux.
- **Browser**: Latest version of Chrome, Firefox, Safari, or Edge.
- **Internet Connection**: Stable internet connection required for multiplayer features.

### Hardware Requirements
- **Processor**: 1.5 GHz or higher.
- **RAM**: At least 4 GB.
- **Storage**: 200 MB of free disk space.

## Installation Instructions
1. **Clone the Repository**: Use `git clone https://github.com/jocabernadas-oss/super-bingo-game.git`.
2. **Navigate to the Project Directory**: Open a terminal and change to the project directory.
3. **Run the Application**: Open the `index.html` file in your web browser.

## How to Play
1. Obtain a bingo card from the lobby.
2. Wait for the game to start and numbers to be called.
3. Mark your card as numbers are drawn.
4. Call out "Bingo!" when you have a winning combination.

## Conclusion
Enjoy playing Super Bingo with friends or players from around the world! For support, please contact our development team.
